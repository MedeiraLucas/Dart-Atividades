# Cartão de Visita Digital - Flutter

## 📌 Descrição do Projeto

Este projeto consiste na criação de um **Cartão de Visita Digital** utilizando o framework **Flutter**.
O objetivo é praticar o uso de widgets básicos da interface gráfica, organização de layout e estilização de componentes.

A aplicação apresenta um cartão com informações profissionais como **nome, profissão, telefone, e-mail e localização**.

---

## 🛠 Tecnologias Utilizadas

* Flutter
* Dart
* Material Design

---

## 📱 Funcionalidades

* Exibição de **nome em destaque**
* Exibição da **profissão**
* Uso de **ícones para informações de contato**
* Organização vertical utilizando **Column**
* Criação de um **card visual usando Container**
* Aplicação de **padding e margin para espaçamento**

---

## 🧱 Estrutura Utilizada

O projeto utiliza alguns widgets principais do Flutter:

* **MaterialApp** → estrutura principal da aplicação
* **Scaffold** → layout base da tela
* **Container** → criação do cartão com fundo e borda
* **Column** → organização vertical dos elementos
* **Text** → exibição das informações
* **Icon** → representação visual dos contatos

---

## 🎯 Objetivo do Exercício

Praticar conceitos fundamentais do Flutter, incluindo:

* Criação de interfaces
* Organização de widgets
* Estilização de componentes
* Uso de ícones e layouts responsivos

---

## ▶️ Como Executar o Projeto

1. Instale o **Flutter SDK**
2. Clone este repositório

```bash
git clone https://github.com/seu-usuario/cartao-visita-flutter.git
```

3. Entre na pasta do projeto

```bash
cd cartao-visita-flutter
```

4. Execute o projeto

```bash
flutter run
```
